# PHP Extras

Some extra PHP snippets for SublimeText.

# Install

## MacOS X

    pushd "~/Library/Application Support/Sublime Text 2/Packages"
    git clone git://github.com/robballou/php-extras-sublimetext.git "PHP Extras"
